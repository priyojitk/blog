1 Insert mode
```
I
```
or 
```
ggVG
```
2 Move all the code to the left side
```
 Shift + <
```
3 Remove trailing space
```
:%s/#*$/    (# represents space; replace with space)
``` 
4 Auto indent 
```
gg=G
```
